Projeto no console

1.Baixe e instale a versão mais recente do Visual Studio Code para seu sistema operacional. ... 2.Inicie o Visual Studio Code. 3.Na barra de ferramentas à esquerda, clique no ícone de extensões. ... 4.Pesquise o Salesforce Extension Pack (Pacote de extensões do Salesforce) e clique em Install (Instalar). 5.Crie uma pasta de arquivos 6.Abra esta pasta com o VS Code e começe a programar!

A tela do console serve para verificar o que está acontecendo no código.
